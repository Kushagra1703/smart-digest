# SmartDigest 🚀
**An AI-powered Research Assistant that summarizes and analyzes documents.**

### 🔗 [Live Demo]([https://smart-digest-frontend-psjyugydj-kushagras-projects-337b5d40.vercel.app/](https://smart-digest-frontend.vercel.app/))

## 📖 Overview
SmartDigest helps users process large amounts of information by providing intelligent summaries and insights. 
Built with a modern tech stack, it features secure authentication, real-time database syncing, 
and high-performance AI processing.

## 🛠️ Tech Stack
- **Frontend**: Next.js 16, Tailwind CSS, Clerk (Auth)
- **Backend**: FastAPI (Python), Google Gemini AI
- **Database**: PostgreSQL (Neon DB), Prisma ORM
- **Storage**: Cloudinary

## ✨ Key Features
- **AI Analysis**: Extract key insights from text and images using Gemini Pro.
- **History Tracking**: Securely save and review your previous research.
- **Cloud Deployment**: Scalable architecture hosted on Vercel and Render.
- **Responsive Design**: Fully optimized for mobile and desktop views.

## 🚀 Local Setup
1. Clone the repository.
2. Install dependencies: `npm install` (frontend) and `pip install -r requirements.txt` (backend).
3. Set up your `.env` variables (see `.env.example`).
4. Run the development servers.
